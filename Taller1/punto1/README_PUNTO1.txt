Este es el Readme de el punto 1, corespondiente a la materia de métodos de simulación.

Al compilar punto1.cpp, se produce el archivo SIR.dat, 
se debe correr el archivo GraficaSIR.gnu (gnuplot GraficaSIR.gnu en la terminal)
se produce el png SIR.png

Al compilar punto1c.cpp, se produce el archivo SIR.dat, 
se debe correr el archivo graficasinf.gnu (gnuplot graficasinf.gnu en la terminal) 
se produce el png s_inf.png
